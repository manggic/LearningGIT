

Commands :

* To list images 
```
 docker image ls 
 docker images
```


* To list running container
```
 docker container ls
 docker ps
```

* To list all container
```
docker container ls -a
```

* To stop container
```
 docker container stop <container-name>
 docker container stop <container-id>
```

* To remove all stopped container
```
 docker container prune
```
